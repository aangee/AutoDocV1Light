#System semi auto-dock
---
Pour les InGameScript
> Gestion partiel de amarage des ship petit bloc

*Le script gère :*
- la connection du connector
- le switch des batteries en mode recharge
- le switch de tanks en mode stockpile
- le switch des thruster ON/OFF
